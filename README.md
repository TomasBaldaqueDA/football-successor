# Football Successor

Football Successor is a data analytics project focused on identifying statistically and tactically suitable player replacements across leagues.

The system builds a multi-season player performance database and applies similarity modelling to detect potential successors for a given player.

## Project Objective

The goal is to design a replacement modelling system capable of:

- Identifying **like-for-like replacements**
- Detecting **role-based alternatives**
- Finding **development prospects**
- Highlighting **market inefficiencies**

The model compares players across leagues using normalized performance metrics and multi-season profiles.

## Data

The dataset was constructed from publicly available football statistics across multiple leagues and seasons.

Primary sources include:

- WhoScored
- Understat

Currently the project includes:

- 12 leagues
- 5 seasons
- ~34,000 player-season observations
